## 📋 Step-by-Step Deployment

### Step 1: Create Railway Account
1. Go to: https://railway.app
2. Click: **"Start for free"**
3. Sign up with GitHub or email

### Step 2: Create New Project
1. Click: **"New Project"**
2. Select: **"Deploy from GitHub"** OR **"Deploy from GitHub Repo"**

### Step 3: Connect Your Repository
Option A - Using GitHub (Recommended):
1. If you haven't already, push this folder to GitHub
2. In Railway, authorize GitHub
3. Select: `AgentWorkspace` repository
4. Select: `telegram-bot-buttons.js` as main file

Option B - Deploy Directly (Easier):
1. In Railway, click: **"Deploy from Git"**
2. Paste your GitHub repo URL
3. Railway will auto-detect and deploy

### Step 4: Add Environment Variables
1. In Railway project, go to: **"Variables"**
2. Add this variable:
   ```
   TELEGRAM_BOT_TOKEN=8581864997:AAEwTweuMOq1nXfsED2-CmeMibzv4JcUB18
   ```
3. Click: **"Deploy"**

### Step 5: Watch It Deploy
1. You'll see logs showing the deployment
2. When complete, it says: **"Running"** ✅
3. Bot is now LIVE 24/7! 🎉
